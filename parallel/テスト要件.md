# テスト要件: omw/parallel

* 要件は、外部から観測できる振る舞いを記述する（内部設計は記述しない）。
* 種別: 期待値=【P. 期待値テスト】 / 統計=【Q. 統計テスト】 / 性質=【R. 性質テスト】
* 作成者: 人間=既存のテストコードから書き起こした要件 / AI=AIが追加した要件

## For（並列ループ）

| ID | 要件 | 種別 | 作成者 |
|---|---|---|---|
| PAR-1 | n個の仕事をp個のワーカーに、連続した範囲で均等に配分する | 期待値 | 人間 |
| PAR-2 | 均等に割り切れない場合、余りはワーカーIDの小さい順に1つずつ配分する | 期待値 | 人間 |
| PAR-3 | p>nの場合、p=nとして扱い、正常に完了する | 期待値 | 人間 |
| PAR-4 | n=0の場合、コールバックを呼ばず正常に完了する | 期待値 | 人間 |
| PAR-5 | nが負の場合、内容が分かるエラーを返す | 期待値 | 人間 |
| PAR-6 | pが1未満の場合、内容が分かるエラーを返す | 期待値 | 人間 |
| PAR-7 | コールバックがエラーを返した場合、そのワーカーは以降の仕事を中断するが、他のワーカーは完走する。返るエラーはerrors.Isで元のエラーと判定できる | 期待値 | 人間 |
| PAR-8 | 複数のワーカーがエラーを返した場合、全てのエラーをerrors.Isで判定できる | 期待値 | 人間 |
