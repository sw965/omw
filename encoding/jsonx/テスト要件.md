# テスト要件: omw/encoding/jsonx

* 要件は、外部から観測できる振る舞いを記述する（内部設計は記述しない）。
* 種別: 期待値=【P. 期待値テスト】 / 統計=【Q. 統計テスト】 / 性質=【R. 性質テスト】
* 作成者: 人間=既存のテストコードから書き起こした要件 / AI=AIが追加した要件

## Save / Load

| ID | 要件 | 種別 | 作成者 |
|---|---|---|---|
| JSN-1 | Saveで保存した構造体をLoadで読み込むと、元の値と一致する（往復の一貫性） | 期待値 | 人間 |
| JSN-2 | UTF-8 BOM付きのJSONファイルも正しく読み込める | 期待値 | 人間 |
| JSN-3 | 存在しないファイルをLoadした場合、os.ErrNotExistと判定できるエラーを返す | 期待値 | 人間 |
| JSN-4 | JSONとして不正な内容のファイルをLoadした場合、エラーを返す | 期待値 | AI |
| JSN-5 | 既存ファイルをSaveで置き換えた場合、新しい値だけが読み込め、一時ファイルが残らない | 期待値 | AI |
