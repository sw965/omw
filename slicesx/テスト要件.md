# テスト要件: omw/slicesx

* 要件は、外部から観測できる振る舞いを記述する（内部設計は記述しない）。
* 種別: 期待値=【P. 期待値テスト】 / 統計=【Q. 統計テスト】 / 性質=【R. 性質テスト】
* 作成者: 人間=既存のテストコードから書き起こした要件 / AI=AIが追加した要件

## Permutations（順列）

| ID | 要件 | 種別 | 作成者 |
|---|---|---|---|
| PER-1 | n個の要素からr個を選ぶ順列を、辞書式の順序で全て列挙する | 期待値 | 人間 |
| PER-2 | r=0の場合、空スライスを1つだけ返す | 期待値 | 人間 |
| PER-3 | r=len(s)の場合、全要素の並び替えを全て列挙する | 期待値 | 人間 |
| PER-4 | rが負の場合、何も列挙しない | 期待値 | 人間 |
| PER-5 | r>len(s)の場合、何も列挙しない | 期待値 | 人間 |
| PER-6 | 入力に重複する要素がある場合も、位置に基づいて列挙する（重複除去しない） | 期待値 | 人間 |

| ID | 要件 | 種別 | 作成者 |
|---|---|---|---|
| SEQ-1 | n個の要素から長さrの重複順列（n^r通り）を全て列挙する | 期待値 | 人間 |
| SEQ-2 | r=0の場合、空スライスを1つだけ返す | 期待値 | 人間 |
| SEQ-3 | len(s)=1の場合、同じ要素をr個並べた1通りだけを返す | 期待値 | 人間 |
| SEQ-4 | rが負の場合、何も列挙しない | 期待値 | 人間 |
| SEQ-5 | sが空の場合、何も列挙しない | 期待値 | 人間 |
| SEQ-6 | 入力に重複する要素がある場合も、位置に基づいて列挙する | 期待値 | 人間 |
