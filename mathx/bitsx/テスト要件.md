# テスト要件: omw/mathx/bitsx

* 要件は、外部から観測できる振る舞いを記述する（内部設計は記述しない）。
* 種別: 期待値=【P. 期待値テスト】 / 統計=【Q. 統計テスト】 / 性質=【R. 性質テスト】
* 作成者: 人間=既存のテストコードから書き起こした要件 / AI=AIが追加した要件

## ビット操作（FromIndices / Size / Bit / Set / Toggle / Clear）

* いずれもuint8とuint64の両方の型で確認する。

| ID | 要件 | 種別 | 作成者 |
|---|---|---|---|
| BIT-1 | FromIndicesは、指定したインデックスのビットを立てた値を返す。空・nilは0を返す | 期待値 | 人間 |
| BIT-2 | Sizeは、型のビット幅（uint8=8, uint64=64）を返す | 期待値 | 人間 |
| BIT-3 | Bitは、指定位置のビット(0/1)を返す | 期待値 | 人間 |
| BIT-4 | Setは、指定位置のビットを1にする（既に1なら変化なし） | 期待値 | 人間 |
| BIT-5 | Toggleは、指定位置のビットを反転する | 期待値 | 人間 |
| BIT-6 | Clearは、指定位置のビットを0にする（既に0なら変化なし） | 期待値 | 人間 |
| BIT-7 | 上記の位置指定操作は、境界（0とビット幅-1）で正しく動作し、範囲外（負・ビット幅以上）ではエラーを返す | 期待値 | 人間 |

## ClearLowest / ExtractLowest / Indices / Singles / IsSubset

| ID | 要件 | 種別 | 作成者 |
|---|---|---|---|
| LSB-1 | ClearLowestは、最下位の1ビットだけを消した値を返す（0は0のまま） | 期待値 | 人間 |
| LSB-2 | ExtractLowestは、最下位の1ビットだけを残した値を返す（0は0のまま） | 期待値 | 人間 |
| LSB-3 | Indicesは、立っているビットの位置を昇順で返す（0は空） | 期待値 | 人間 |
| LSB-4 | Singlesは、立っているビットを1つずつ分離した値を下位から順に返す（0は空） | 期待値 | 人間 |
| LSB-5 | IsSubsetは、subの全ビットがsuperに含まれる場合にtrueを返す（等しい場合・両方0もtrue） | 期待値 | 人間 |

## Matrix（生成・ビット操作）

| ID | 要件 | 種別 | 作成者 |
|---|---|---|---|
| MTX-1 | NewZerosMatrixは、指定形状の全ビット0の行列を返す。行数・列数が0以下の場合はエラーを返す | 期待値 | AI |
| MTX-2 | NewOnesMatrixは、論理的な全ビット（rows×cols個）が1の行列を返す（列数が64の倍数でなくても、OnesCountはrows×colsになる） | 期待値 | AI |
| MTX-3 | Set/Bit/Clear/Toggleは、指定した(行, 列)のビットを操作・取得できる。範囲外はエラーを返す | 期待値 | AI |

## Matrix（転置・距離・内積）

| ID | 要件 | 種別 | 作成者 |
|---|---|---|---|
| MTX-4 | Transposeは、形状が入れ替わり、全ての位置で元の(r,c)と転置後の(c,r)のビットが一致する。二回転置すると元に戻る。様々な形状（1x1、64の倍数、端数あり）で成り立つ | 性質 | AI |
| MTX-5 | HammingDistanceは、自身との距離が0、全ビット反転との距離が総ビット数になる。形状不一致はエラーを返す | 期待値 | AI |
| MTX-6 | Dotの結果は、素朴な参照実装（一致ビット数の計算）と、様々な形状で完全に一致する。高速化実装（AVX-512）とフォールバック実装の両方で成り立つ | 性質 | 人間 |
| MTX-7 | DotTernaryの結果は、素朴な参照実装（符号一致数−不一致数）と、様々な形状で完全に一致する。両方の実装で成り立つ | 性質 | 人間 |
| MTX-8 | HammingDistanceの結果は、XORとOnesCountによる参照実装と一致する。両方の実装で成り立つ | 性質 | 人間 |
